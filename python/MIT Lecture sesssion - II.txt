=========================================

* In Python every function is an object. And function's name 
is reference.
* This is the reason, Python does not support method overloading.

* In Python, we have a py-documentation ---> __doc__
* This will give us any help/documentation related to any 
function, module, class, library....

For example:
	print(print.__doc__)
This will print documentation of print() function.

	print(str.__doc__)
This will print documentation of class str.

* We can check datatype of any entity, by using type().

For example:
--------------
a  = 10
print(type(a))

a = 9.2
print(type(a))

* In Python, we don't have any primitive datatypes. All 
datatypes are class names.

	- int
	- float

* In Python, we have 4 data structures.
	- list
	- tuple
	- set
	- dict (dictionary)

* To input a value in Python, we have input() function.
* This function obtains and returns input value in form of 
string. 
* Therefore we have to parse input string into required datatype.
* To parse, we have constructors of class.

For example:
-------------
a = int(input())
print(type(a))

* Here, input() is taking input and returning as stirng.
* The int() constructor is converting that string into "int"

For example:
-------------
print("Enter age of student")
a = int(input())

if a>18:
    print("Eligible for Voting")
else:
    print("Not eligible for Voting")

	** 	to obtain power
	%	to obtain remainder
